### 这是一个在查看官方插件并且学习过程，改写的插件

修复内容：  

1.打开一个场景默认拥有多个平行光  

2.场景绘制闪烁  

3.场景中粒子并没有正常显示  

4.默认情况下会有多选状态，并不会自动清除选择标记  

5.默认采用lumen作为首选的全局光照造成性能浪费  

6.默认选择最后点击的作为选择项目，不是按住鼠标表示拖拽选项.与编辑器操作不够统一  

8.不会显示“Billiboard Sprites”,选择上的不方便  

9.勾选“RealTime"并不会真的实时
